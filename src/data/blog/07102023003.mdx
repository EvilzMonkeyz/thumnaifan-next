วันที่ 25 ก.ย. เวลา 18.00 น. ที่หาดพยูน อำเภอบ้านฉาง จังหวัดจังหวัดระยอง มีเรื่องมีราวราวความเชื่อชวนขนหัวลุก จากที่มีเสียงร่ำลือถึงวิญญาณเฮี้ยนเพศหญิงสวมสไบมาปรากฏร่างให้มองเห็นก่อนให้ถูก โชครางวัลลอตเตอรี่ 2 ตัวด้านหลังของงวดก่อนหน้าที่ผ่านมา

นักข่าวลงพื้นที่พิสูจน์ตามเสียงร่ำลือเจอนาย นายสาคร ทองเกลี้ยง อายุ 70 ปี ชาวประมงประจำถิ่นได้พาเดินลัดไปตามหาดจนถึงไปพบต้นจามจุรีขนาดใหญ่ถูกคลื่นซัดล้มลงในสมุทร เจอศาลเพียงตามีชุดไทยกองอยู่ที่โคนลำต้นทั้งยังมีถ้วยจานของเส้นไหว้วางเกลื่อนกลาดเรี่ยราดเรี่ยทั่วรอบๆที่มีคลื่นลมซัดเข้าพบริมฝั่งโรงแรมร้าง

![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1160194327809642569/0ccd3af9255f61f1.jpeg?ex=654f75ab&is=653d00ab&hm=35fa4d089d63ffca5f821b30c683cb3bbd492ccafc0bcd5e24150bb013eca01c&=&width=1210&height=681)

นายนิยม เสมา อายุ 73 ปี เจ้าหน้าที่รัฐเกษียณซึ่งอาศัยอยู่ในพื้นที่ได้เล่าลี้ลับถึงความฝันที่เจ้าแม่แดง มาฝันให้มองเห็นเป็นหงส์ตัวใหญ่สีขาวบินมาในสมุทรซึ่งตอนนั้นตนได้คุยกับเพื่อนฝูงที่เป็นผู้กำกับว่าให้ใช้ปืนยิงกระทั่งนกหงส์ตกลงมาในสมุทรแล้วตนก็เข้าไปกอดจับเจอเป็นหญิงสาวงามกระทั่งมีการเสวนากันรวมทั้งให้เลขเด็ดมาเป็นเลข 305 ในหวยรัฐบาลงวดที่จะถึงนี้

น.ส สุวาทนี จันทรสุข อายุ 46 ปี แม่ค้าลูกชิ้น รับรองว่าศาลที่นี้เก่าและก็มีเรื่องมีราวราวความเฮี้ยนของเจ้าแม่แดงมานานซึ่งปัจจุบันรับโชคกันไปเต็มๆจากเลขท้าย 2 ตัวหวยรัฐบาลงวดก่อนหน้าที่ผ่านมา

![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1160194328325537833/47d85452b50cdb5b.jpeg?ex=654f75ab&is=653d00ab&hm=02ad338346d0a3c19f7337c25387936f84e04fee31acb44808da263678830132&=&width=1210&height=565)

นายนิว พ่อค้าลูกชิ้นทอด อีกร้านค้าหนึ่ง เล่าว่าสมัยก่อนตัวเอง เคยเป็นบุคลากรของโรงแรมที่เลิกกิจการไปหลายสิบปี ในขณะนั้นเห็นมากับตาพร้อมเพื่อนฝูงอีก 2 คน มองเห็นเป็นเพศหญิงงามสวมสไบเดินผ่านให้มองเห็นแบบจะๆจนกระทั่งจำต้องรีบวิ่งหนีทำเอาหลอนกันไปอีกทั้งชุมชนอย่างยิ่งจริงๆด้วยเหตุว่าไม่ได้มองเห็นเพียงแค่ครั้งเดียวมาให้มองเห็นบ่อย

อย่างไรก็แล้วแต่จากการลงพื้นที่ตรวจสอบแล้วก็พบว่าต้นจามจุรีขนาดใหญ่ถูกคลื่นสมุทรซัดจนกระทั่งทำให้เป็นต้นเหตุให้ตายลงส่วนเรื่องราวความเชื่อถือก็ว่ากันไปในส่วนของประชาชนบางคนที่เขาอยู่กันมาพร้อมกับวิญญาณหญิงสาวงามสวมสไบที่ไปพบเห็น
