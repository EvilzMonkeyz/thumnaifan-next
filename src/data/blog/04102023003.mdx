นางเอกสาว ขวัญ อุษามณี ยกให้เธอเป็นสาวเก่งอีกคนของแวดวงบันเทิงที่นอกเหนือจากที่จะเล่นละครเป็นอาชีพที่เธอรักแล้ว ปัจจุบันนี้ ขวัญ ยังเป็นนักธุรกิจอีกด้วยเปิดแบรนด์เครื่องสำอางค์ของตนเองมีหลากหลายสินค้า

จะต้องพูดว่ายอดจำหน่ายปังสุดๆออเดอร์ล้นจริงๆเป็นอีกหนึ่งธุรกิจที่คุณสร้างมาด้วยความรู้ความเข้าใจและก็วันนี้ไปถึงเป้าหมายแล้ว

![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1159156966401253486/6a9b624dc70752f5.jpeg?ex=6542750c&is=6530000c&hm=3e92b6f0f8f5d849d2d1a56f2620a0981a7109e3f3a18db6df69042090a543db&=)

ปัจจุบัน ขวัญ อุษามณี จัดของขวัญชิ้นใหม่ให้กับตนเองอีกครั้งกับการซื้อรถยนต์เบนซ์คันหรูไว้นั่งไปดำเนินงาน ถือฤกษ์ดีไปรับรถยนต์เป็นที่เรียบร้อย ราคา 5 ล้านปลายๆยังไม่รวมแต่งด้านในอีกนะเนี่ย

ซึ่งเธอได้เขียนเล่าไว้ว่า "ลูกรักมาแล้ววววว ตั้งชื่อว่าพี่อะไรดีจ๊ะ รับทรัพย์สมบัติรับความสุขไปร่วมกันนะ พร้อมปฏิบัติงาน ลุยสุดกำลังค่า ให้อุษาทำอะไรก็ได้เว้นเสียแต่ทำไม่สวย ...อันนี้ทำยากจริงๆ555 อุษาทำไม่ได้"

![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1159156967034605598/3.jpeg?ex=6542750d&is=6530000d&hm=0f526ffac9332a812af5b8624596e91fd40e2f2bffc89d10a14b71a946d4b149&=)

เป็นรถยนต์ที่เธอไว้ใช้ดำเนินการไปรับเงินรับทอง ผู้คนจำนวนมากมองเห็นแล้วต่างเข้ามาร่วมให้พรยินดี พร้อมด้วยแห่ถามเลขทะเบียนกันมากมายเลยนะ
