summary: ชาวบ้านเจอเต่าปริศนารูปร่างแปลก มาเขย่าพุ่มหน้าบ้าน พอเดินเข้าไปมองกลับมีเสียงขู่ฟ่อๆเหมือนเสียงงู แต่ว่าไม่ดุร้าย ก่อนเดินเชื่องช้าเข้าบ้านเฉย ในช่วงเวลาที่นักเสี่ยงดวงไม่พลาด แห่ขอหวยกันต่อเนื่อง

วันนี้ (8 ตุลาคม) นักข่าวได้รับแจ้งจากประชาชน ตำบลโคกปรง อำเภอวิเชียรบุรี จังหวัดจังหวัดเพชรบูรณ์ ว่า มีคนเจอเต่าปริศนา มีรูปร่างลักษณะพิศดาร ต่างจากเต่าที่เคยพบเจอทั่วๆไป โดยที่กระดองของเต่าตัวนี้จะเรียบไม่โค้งมนรวมทั้งมีลวดลายสีดำ ส่วนขอบชายกระดองไม่บานราวกับเต่าทั่วๆไป มาเขย่าพุ่มหน้าบ้านของชาวบ้านจนสั่นไหว

กระทั่งเจ้าของบ้านนึกว่าผีหลอก พอเดินเข้าไปมองใกล้ๆกลับมีเสียงขู่ฟ่อๆเหมือนเสียงงู แต่ไม่ดุร้ายเดินกระต้วมกระเตี้ยมเข้าบ้านเฉย คล้ายกับอยากมาขออาศัยอยู่ด้วยกับคน เจ้าของบ้านก็เลยจับมาเลี้ยงไว้ โดยมั่นใจว่าจะนำโชคมาให้


![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1161297601782616125/1.jpeg?ex=6553792c&is=6541042c&hm=90e361e2601614ce0dc45bd38b0d01a8904a2746fc0c137e681cbaf5264faea6&=&width=1210&height=681)

ในเวลาที่เพื่อนบ้านที่รู้ข่าวสาร ต่างแห่กันมาขอดูและไม่พลาดที่จะถามไถ่เลขที่บ้านแล้วก็ส่องหาเลขเด็ด เพื่อจะนำไปเสี่ยงดวง งวดวันที่ 16 ตุลาคม2566 นี้ ก็เลยเดินทางไปตรวจดูที่บ้านเลขที่ 85 หมู่ 17 ตำบลโคกปรง อำเภอวิเชียรบุรี จังหวัดจังหวัดเพชรบูรณ์

จากการสอบถาม นายตาว กันหาเขียว อายุ 66 ปี เจ้าของบ้าน และก็เป็นผู้เจอเต่าแปลกตัวดังที่กล่าวมาแล้วข้างต้น ได้กล่าวมาว่า เมื่อคืนนี้ตอน 2 ทุ่ม ระหว่างที่ตนกำลังเดินเที่ยวอยู่หน้าบ้าน ได้มองเห็นใบไม้ก้านไม้ที่อยู่ริมทางเขย่าเสมือนมีอะไรแอบอยู่ด้านในพุ่ม ทั้งๆที่ก็ไม่มีลมพัด


![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1161297602294333622/3.jpeg?ex=6553792c&is=6541042c&hm=f5466673231304d6c5a30160c73ca4f4b8807ae6014a8f3b51655c4846b774bf&=&width=1210&height=681)

ตอนแรกรู้สึกว่าถูกผีหลอก แต่ตนเป็นผู้ที่ไม่กลัวผี ก็เลยได้เดินเข้าไปมองใกล้ๆแต่ว่าก็ยังไม่เห็น ก็เลยได้ใช้ไฟฉายส่องมองกับได้ยินเสียงขู่ ดังฟ่อๆเสมือนเสียงงู แต่ตนเห็นว่ามันไม่ใช่งู ก็เลยได้แหวกต้นไม้ดูและพบว่าเป็นเต่าที่มีรูปร่างหน้าตาที่พิศดารจากเต่าปกติ มองได้จากกระดองที่มีลักษณะเรียบไม่โค้งมนรวมทั้งขอบกระดองไม่บานราวกับเต่าทั่วๆไป

และก็ยังมีน้ำหนักตัวที่มากถึง 8 กิโล 9 ขีด ซึ่งตามเดิมที่เคยได้เห็นเต่าตัวขนาดนี้จะมีน้ำหนักราวๆเพียงแค่ 4-5 กิโล แต่เต่าตัวนี้หนักมาก แล้วก็ตนก็มั่นใจว่าเต่าตัวนี้จำเป็นต้องนำโชคมาให้ครอบครัวของตนเองอย่างไม่ต้องสงสัย ด้วยเหตุว่ามีเลขหลายตัวอยู่ที่กระดอง ดังเช่น 78 - 85 โดยเฉพาะอย่างยิ่งเลข 85 ซึ่งก็ตรงกับเลขที่บ้านของตัวเอง
