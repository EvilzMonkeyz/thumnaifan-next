เมื่อเวลา 10.30 น. วันที่ 2 ตุลาคม2566 นักข่าวกล่าวว่า มีผู้เสียหายเป็นผัวเมีย เข้าแจ้งเหตุลงบันทึกประจำวันไว้ที่สภ.ตลุกดู่ จังหวัดจังหวัดอุทัยธานี ภายหลังที่ซื้อหวยผ่านทางออนไลน์ ที่มีเพจชื่อว่า เฮง เฮง ลอตโต้ โดยซื้อขายแลกเปลี่ยนในวันที่ 23 เดือนกันยายน2566 โดยมีหลักฐานการค้าขายทางแชท เจ้าหน้าที่ก็เลยลงบันทึกประจำวันไว้เป็นหลักฐาน

จากการซักถามนายสมเกียรติ เลี่ยมสุวรรณ อายุ 60 ปี พร้อมด้วยนางสาวนารีรัตน์ ภันทมร อายุ 37 ปี ซึ่งเป็นเมียของนายสมเกียรติ ได้เล่าเหตุ ในวันซื้อหวยทางออนไลน์กับเพจ เฮง เฮง ลอตโต้ พร้อมด้วยเล่าเรื่องให้ฟัง

![enter image description here](https://cdn.discordapp.com/attachments/1158269424156028998/1158314864549302362/30.jpg?ex=65362a48&is=6523b548&hm=959d0b6628b80f2f4f8ed3d847a4ce18ebcf8bb7d4f071f40a724d2799dc6383&)

โดยนำหลักฐานสำหรับการค้าขายมาให้ ผู้รายงานข่าวดู ใน วันที่ 23 เดือนกันยายน 2566 การค้าขายทางออนไลน์เลข 727202 โดยเก็บเงินที่จุดหมายปลายทางจำนวนเงิน 720 บาท ส่วนใบลอตเตอรี่นั้นยังอยู่กลับคนขายทางออนไลน์ โดยที่ยังไม่ส่งมาให้ ต่อจากนั้นทางเพจ ในวันถัดมา ก็ยังเปิดเพจธรรมดา จนกระทั่งมาวันสลากกินแบ่งออก พอหลังจากสลากกินแบ่งออกผล ตนเองก็เลย มาตรวจสอบเลขก็ดันถูกรางวัลที่ 1 ปริมาณ 5 ใบ เป็นเงินราวๆ 30 ล้านบาท

![enter image description here](https://cdn.discordapp.com/attachments/1158269424156028998/1158314864817745990/3db3cfc74dc68bbc.jpg?ex=65362a48&is=6523b548&hm=7549813af1d8b96c2d4157749a8dc4616f03587e32a8d56ccd88c12ecdf08da8&)

แล้วตัวเองติดต่อไปทางเพจ แต่ว่าถูกทางคนขายอ้างกลับมาว่าทางคนซื้อไม่ตอบกลับรวมทั้งติดต่อมิได้เพราะเหตุว่าจำนวนที่ได้ค้าขายทางออนไลน์นั้นมาจากการประกาศไหว้เขย่าเซียมซีกับหลวงพ่อกวย จังหวัดจังหวัดชัยนาท ต้นเหตุในคราวนี้ทางคนขายได้กล่าวถึงว่าทางผู้บริโภคมิได้ติดต่อกลับมาก็เลยเกิดเหตุการณ์ดังที่กล่าวมาแล้วข้างต้นให้ 2 ผัวเมียเข้าแจ้งเหตุบันทึกประจำไว้เป็นหลักฐาน
