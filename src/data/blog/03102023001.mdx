วันที่ 3 เดือนตุลาคม 2566 นักข่าวกล่าวว่า เมื่อคืนนี้ก่อนหน้านี้ ที่วัดบ้านเบง หมู่ที่ 2 ตำบลศรีณรงค์ อำเภอชุมพลบุรี จ.จังหวัดสุรินทร์ ได้มีชาวบ้านจับกลุ่มกันมาขอเลขเด็ดต้นข่อยอายุยงหลายร้อยปี ที่อยู่ด้านในวัดบ้านเบง ซึ่งชาวบ้านมั่นใจว่าคงจะมีสิ่งศักดิ์สิทธิ์อาศัยอยู่ โดยก่อนหน้าที่ผ่านมาชาวบ้านเคยมาขอโชคลาภจากต้นข่อยต้นนี้ และก็ให้ลาภถูกหวยกันมาหลายรอบแล้ว

![enter image description here](https://cdn.discordapp.com/attachments/1158269424156028998/1158749149810937956/3.jpg?ex=6540f93d&is=652e843d&hm=0c1a2ff162dc755cd6c5da0d0160b7d1774082a2b622309e0b0ad6987a0eede9&)

โดยปัจจุบันเมื่องวดวันที่ 1 เดือนตุลาคม66 ก่อนหน้านี้ที่ผ่านมา ซึ่งมีประชาชน (ไม่ต้องการออกนาม) เข้ามาขอเลขโดย ใช้ธูปเสี่ยงทายเลข ปรากฎได้เลข 69-66 ก่อนนำตัวเลขไปหาซื้อสลากกินแบ่งได้เลข ด้านหลัง 66-69 หลายใบ พร้อมทั้งบนไว้ว่า ถ้าหากว่าถูกลอตเตอรี่จะนำหนังกลางแปลงมาฉายให้มอง รวมทั้งปรากฏว่าสลากกินแบ่งงวดวันที่ 1 ต.ค. 2566 โดยรางวัลอันดับแรกออก 727202 ส่วนรางวัลเลขท้ายสองตัว ออก 66 ปรากฏว่าตรงกับรางวัลที่ซื้อไว้ รวมทั้งถูกรางวัลหลายใบ ได้เงินหลัก 100,000 บาท ก็เลยได้มีการว่าจ้างหนังกลางแปลงมาฉายแก้บนข้างๆต้นข่อยดังกล่าว ตามข้อตกลงที่ได้เคยให้ไว้ก่อนหน้าที่ผ่านมา ท่ามกลางบรรยากาศด้านในวัดที่สุดแสนจะวังเวง

![enter image description here](https://cdn.discordapp.com/attachments/1158269424156028998/1158749150062587924/89210b602a650970.jpg?ex=6540f93d&is=652e843d&hm=d679edbef5ab359155dcbc5364d621c60a2f28bd8b0f84eca1ca91945a3138cc&)

ในงวดถัดไปที่จะมาถึงนี้ ชาวบ้านได้ใช้ธูปเสี่ยงทายอีกเหมือนปกติ โดยจำนวนที่ได้ก็คือ 385 ซึ่งประชาชนจะได้นำตัวเลขที่ได้นี้ นำไปเสี่ยงดวงซื้อลอตเตอรี่ ในงวดวันที่ 16 เดือนตุลาคม 2566 ที่จะถึงนี้ถัดไป โดยทุกคนหวังว่าสิ่งศักดิ์สิทธิ์ที่อาศัยอยู่ในต้นข่อยร้อยปี จะให้ลาภให้โชคกับชาวบ้านทุกงวดถัดไปอีก
