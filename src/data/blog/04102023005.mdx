การจัดสร้างเหรียญพระเครื่องลาง รุ่นเฮง 168 ดังที่ได้กล่าวมาแล้วข้างต้น จัดสร้างโดย เต็งสี่ พระใหม่ แล้วก็ กร รถขนพระ ซึ่งในเวลาเช้า เวลาโดยประมาณ 12.09 น. ได้มีพิธีการพราหมณ์ บวงสรวงสิ่งศักดิ์สิทธิ์ รวมทั้งหลังจบพิธีการ ได้มีการจุดประทัด เจอเลขหางประทัด กลับตรงกันกับรุ่นของวัตถุบูชา เป็นรุ่น 168 อีกด้วย โดยเลขหางพลุได้เลข 943 และก็ 68

![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1159159347423744071/19dcdc2a8c26860c.jpg?ex=654bb1c4&is=65393cc4&hm=c250c676f6d5940732ca64105abb88c58e1c862a6ec90defa2f374abed65d07a&=&width=1210&height=681)

ทำให้เป็นที่ฮือฮาแก่ผู้มาร่วมงานเป็นอย่างยิ่ง ที่ต่างพากันไปมุงดูและถ่ายรูปบันทึกไว้ ซึ่งคอลอตเตอรี่ก็ไม่พลาดที่จะนำเลขเด็ดดังกล่าวมาแล้วข้างต้นไปเสี่ยงดวงในงวดที่จะถึงนี้เหมือนปกติ ในช่วงเวลาที่พ่อค้าแม่ค้าหวยก็ขายดิบขายดีเป็นเทน้ำเทท่าไปตามๆกัน ทำให้เลข 68 ถึงกับหมดแผงอย่างยิ่งจริงๆ

![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1159159346161266748/05b2c92a8fe83c53.jpg?ex=654bb1c4&is=65393cc4&hm=f868fff2fcdaa9328f77475b82fcaf0d89fb23910b847b597dc0c722bc14c072&=)

ต่อจากนั้นในตอนบ่าย หลวงปู่เฮง ปภาโส เจ้าอาวาสวัดปรับปรุงธรรมาราม เป็นประธานฝ่ายพระสงฆ์ และก็หลวงตาแก่น คัมภีรมุนี เจ้าอาวาสวัดสุวรรณรัตน์โพธิยาราม ร่วมในพิธีการพุทธาภิเษก นั่งปรกอธิษฐานจิตปลุกเสกวัตถุบูชา เหรียญพระเครื่องลางรุ่นเฮง 168 อย่างเข้มขลังค์ ท่ามกลางประชากรชาวพุทธในพื้นที่ชายแดนไทย-เขมรและก็จากบ้านนอกที่มาร่วมพิธีการแล้วก็รอคอยรับแจกเหรียญพระเครื่องลางนับพันคน
