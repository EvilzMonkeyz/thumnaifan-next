ฮือฮากันไม่ใช่น้อย เมื่อเจ้าแม่บอกหวยคนมีชื่อเสียง อย่าง แม่น้ำหนึ่ง ภิรดา ธนโชติจินดา ผู้ครอบครองวลีได้รับความนิยม "บุญนี้บารมีปู่" ที่ชอบออกมาบอกแนวทางหวยให้เหล่าคนรักตัวเลขได้ลุ้นตรวจสลากกินแบ่งไปร่วมกันในทุกๆงวด

ปัจจุบัน แม่น้ำหนึ่ง ก็สร้างความฮือฮาให้กับเหล่าคอสลากกินแบ่งได้แตกตื่นกัน เมื่อเจ้าตัวได้โพสต์ภาพลงในเฟซบุ๊ก ภิรดา ธนโชติจินดา พร้อมทั้งคู่รัก ยืนคู่กันหน้ารถยนต์หรู ปอร์เช่ คาร์เยน (Porsche Cayenne) สีขาว พร้อมเนื้อความบอกว่า

![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1162393084617818252/79f38d27ccbeaff6.jpg?ex=6557756b&is=6545006b&hm=cf0d3e1d4551f1b3feda6aeb3bea10d7a31d50866b35595d7f4185048f2ce55f&=&width=421&height=749)

"บันทึกไว้ ว่าฉันได้ปฏิบัติตามความฝันเสร็จแล้ว ขอบพระคุณผู้อยู่เบื้องหลังนะคะ พงษธร คงเพ็ชรศักดิ์ คิดสิคิดสิ คาปูชิโน่ ขอบคุณมากรถยนต์งามๆจาก Munggon Used Car Premium ขายรถยนต์ยุโรปทุกหมวดหมู่จังหวัดร้อยเอ็ด สำนักงานใหญ่"

เพียงพอมองเห็นรูป แม่น้ำหนึ่ง ออกรถยนต์หรูแบบงี้ ไอ้คนรักลอตเตอรี่อย่างพวกเราก็ว้าวุ่นเลย รีบเลื่อนลงมามองเลขเด็ดจากทะเบียนรถยนต์แม่น้ำหนึ่งอย่างไว แม้กระนั้นหาใช่เช่นนั้นไม่ ดันมีสติ๊กเกอร์เจ้ากรรมปิดไว้ หมดกันเลขเด็ดงวดนี้ที่วาดฝัน !
