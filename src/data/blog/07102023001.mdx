จากกรณีมีเศรษฐีจากจังหวัดจันทบุรี เป็นผู้ใจบุญมอบรถยนต์เบนซ์ป้ายแดง ราคา 2,540,000 บาท แก่พระคูรอุดมปัญญาสาร วัดบ้านหนองเรือ ตำบลหนองหลวง อำเภอเสลภูมิ จังหวัดร้อยเอ็ด ซึ่งเป็นพระอาจารย์ภาคอีสานที่เป็นที่รู้จักมีชื่อเสียงด้านเมตตา รวมทั้งท่านเป็นลูกศิษย์หลวงปู่คูณที่วัดบ้านไร่เมื่อสมัยก่อน

![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1160174166167801917/797bd16dde175ec5.jpg?ex=654f62e4&is=653cede4&hm=a07d00bd4e41b5116dfec99018d7a14e2dbbdc2bea15e340ad84a18a097a2ffb&=)

ทำให้เป็นที่ฮือวานนี้ (6 ตุลาคม66) กรรมการวัด รวมทั้งศิษยานุศิษย์ ด้วยกันประกอบพิธีสวดมนตร์เจริญพระพุทธมนต์ เพื่อความมงคล แจ้งเทวดาฟ้าดิน สิ่งศักดิ์สิทธิ์ทั้งหลายแหล่ในวัด สำหรับในการรับมอบรถยนต์ เบนซ์ป้ายแดง 1 คัน เลขทะเบียน 6520 กทม

![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1160174166545285202/2.jpg?ex=654f62e4&is=653cede4&hm=5f4865847a0da18fe7f5ea7c70b4bce6817d095f7583d4577e3fc76dfb39af35&=&width=1210&height=681)

ซึ่งบรรดาคอลอตเตอรี่ที่มาร่วมงานพากันส่องเลขทะเบียนรถยนต์ที่เอามามอบให้ เพื่อนำไปเสี่ยงดวงแทงหวยงวดวันที่ 16 เดือนตุลาคม 2566 มั่นใจว่าสิ่งศักดิ์สิทธิ์ที่มาร่วมคงจะให้ลาภราวกับทุกครั้งที่ทางวัดมีการประกอบพิธี

ด้าน ประชาชนที่มาร่วมงาน กล่าวมาว่า ตนมาจากบ้านนาแซง อำเภอเสลภูมิ จังหวัดร้อยเอ็ด สาเหตุที่ซื้อเลขตัวนี้ เพราะเหตุว่ารู้สึกว่าคงจะนำโชครวมทั้งฝันมองเห็นเลขตัวนี้ด้วย ฝ้นว่ามีคนมาบอก "ซื้อเลขตัวนี้เด้อ!" 95 กับ 20 ได้ตัวเดียวกันกับเลขรถยนต์ที่เขาเอามามอบให้
