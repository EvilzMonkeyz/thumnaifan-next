ไม่พลาดเลยสักงวดเลยจริงๆสำหรับนักร้องลูกทุ่งสาว ฮาย-อาภาพร นครสวรรค์ ที่ชอบซื้อหวยไม่น้อยเลยทีเดียว และก็เดินสายหาเลขเด็ด รวมทั้งโค้งสุดท้ายแบบงี้ ฮาย อาภาพร ก็ไม่มีพลาดแจกเลขเด็ดให้แฟนคลับกันอีกแล้ว

![enter image description here](https://cdn.discordapp.com/attachments/1158269424156028998/1158759871420825630/20f23a0b873f5169.jpeg?ex=6541033a&is=652e8e3a&hm=8ee9ae4d52cc99d42413fb9b4cdb7a9eb8597ac5369c27af47e772598198fab3&)

โดยปัจจุบัน (30 เดือนกันยายน) ก่อนวันสลากกินแบ่งออกเพียงแค่หนึ่งวัน ฮาย อาภาพร ได้โพสต์ภาพถือกระถางที่เอาไว้สำหรับใส่ธูป พร้อมเอาอกเอาใจแฟนคลับที่ถูกใจเสี่ยงดวง โชว์เลขเด็ดจากเลขธูป 3 ตัวตรง ให้ส่งกันบบชัดๆแล้วก็แคปชั่นที่เขียนไว้ว่า "วันพรุ่งมั่งมีนะคะทุกคนค่ะ #อาภาพรนครสวรรค์ #อาภา #ฮายอาภาพรนครสวรรค์ #สวรรค์อาภา #ภาหาเลข"

![enter image description here](https://cdn.discordapp.com/attachments/1158269424156028998/1158759871710236752/a525e45673c70c27.jpg?ex=6541033a&is=652e8e3a&hm=8f39c5cc96c317e6618666dd9809166a9b8f0bbd332e7aa9882bdcc030735828&)

ทำเอาแฟนคลับเข้ามาสาธุกันใหญ่ พร้อมน้อมรับคำให้พรว่าขอให้มั่งคั่งๆและก็เข้ามาขอบคุณมาก อาภาพร อย่างเช่น แม่ฮายสวยมากมาย เสร็จจากการแสดงดนตรีมาเหน็ดเหนื่อยๆแล้ว ยังมาจุดธูปให้ fc อีก,ควรจะตามโพสต์นี้หรือโพสต์ก่อนหน้าดีค่ะ ต้องการมีโชคมีลาภบ้างจ้ะ ช่วยชี้แนะหน่อยนะคะ,น้อมรับเลขมั่งคั่งไปกับพี่ฮายด้วยคนจ้ะ Yeah Yeah,สาธุขอให้มีโชคจากคุณฮายด้วยเถิดขอบพระคุณนะคะ,ขอบคุณมากจ้ะสาธุๆๆๆๆๆจ้ะ,ขอบพระคุณค่ะพี่ฮายแล้วก็คุณยาย วันพรุ่งร่ำรวยกันทุกคนนะคะ,เลขธูป 622/146 /22/16/61/26/62/29/92 งวด01 เดือนตุลาคม66 ปังๆๆๆจ้ะ,สาธุจ้ะ..มั่งคั่งๆโชคดีๆจ้ะ,เลิกซื้อมานานมากแล้วงวดนี้ทดลองเสี่ยงดูหากมีโชคร่วมทำบุญสุนทานด้วยจ้ะสาธุขอให้มีโชค,ผ้า9สี9ศอกพร้อมละ
