นักข่าวกล่าวว่า ได้เดินทางไปยังบ้านเลขที่ 79 หมู่ 2 บ้านนาทราย อำเภอนาตาล จังหวัดจังหวัดอุบลราชธานี หลังได้รับแจ้งว่ารากไม้ตะเคียน ลักษณะซึ่งคล้ายหัวพญานาค ลอยมากับแม่น้ำโขง พร้อมเรือไม้ตะเคียนโบราณ

![enter image description here](https://cdn.discordapp.com/attachments/1158269424156028998/1159155733212966953/2.jpg?ex=654273e6&is=652ffee6&hm=15555dd225fb937444310c4d0e0f57743ab3b1076d64b25825515725b011d987&)

เมื่อไปถึง นายสนิท บุญทด อายุ 58 ปี เจ้าของบ้าน พานักข่าวไปดูรากไม้ตะเคียน ลักษณะก็จะคล้ายหัวพญานาค รวมทั้งเรือไม้ตะเคียน ที่เอามาไว้รอบๆริมฝั่งแม่น้ำโขงข้างบ้าน โดยได้นำศาลเจ้ามาตั้ง ให้ประชาชนได้มาขอโชคลาภ

![enter image description here](https://cdn.discordapp.com/attachments/1158269424156028998/1159155733607223307/3.jpg?ex=654273e6&is=652ffee6&hm=1caae2e53b13bf4290793a02741a17483bf6658d77ece31202fa4bd1db821ca7&)

นายสนิท พูดว่า รากไม้ตะเคียนลักษณะที่คล้ายหัวพญานาค แล้วก็เรือไม้ตะเคียนได้ลอยมากับแม่น้ำโขง ซึ่งลอยมาอยู่ริมฝั่งข้างบ้านตัวเอง ราษฎรที่หาปลารอบๆดังที่กล่าวมาแล้วมองเห็นก็เลยพากันนำขึ้นมาไว้ ก็เลยขอชื้อกับประชาชนเพราะเหตุว่าถูกใจ ธรรมดาเวลาน้ำโขงขึ้น ไม้จะไหลมาตามน้ำ ตัวเองถูกใจรักษาของโบราณก็เลยซื้อเก็บไว้

ภายหลังที่ได้มาก็มีแต่ความเจริญก้าวหน้า ชีวิตความเป็นอยู่ดียิ่งขึ้นกว่าแต่ก่อน และก็ที่สำคัญเคยให้ลาภให้โชคอยู่เป็นประจำ หากขอก็เป็นได้ ราษฎรรู้ข่าวสารก็มาจุดธูปขอโชคลาภไม่เคยขาด ก็ถูกหวยประจำ สำหรับงวดนี้ขอได้เลขเด็ด 79,09 เพื่อจะไปซื้อสลากกินแบ่งลุ้นรวย
