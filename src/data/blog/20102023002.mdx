วันนี้ (20 ตุลาคม) เมื่อเวลา 15.30 น. อาสากู้ภัยมูลนิธิส่งเสริมธรรม จังหวัดจังหวัดอุดรธานี ได้รับแจ้งมีงูอยู่ในรถเก๋ง ซึ่งเจ้าของขับเข้าไปหยุดในปั๊มน้ำมัน แห่งหนึ่งถนนหลีกเลี่ยงเมือง ในเขตเทศเมืองหนองสำโรง อำเภอเมือง จังหวัดจังหวัดอุดรธานี ก็เลยนำอุปกรณ์จับงูออกไปตรวจสอบ

พบรถเก๋งมิตซูบิชิ มิราจ สีดำ เลขทะเบียน 8901 จังหวัดนครพนม หยุดอยู่ที่จอดรถปั๊มน้ำมัน มี นายประยงค์ อายุ 67 ปี ชาวบ้านหมู่ 3 ตำบลสามพร้าว อำเภอเมือง จังหวัดจังหวัดอุดรธานี เจ้าของรถยนต์ ชี้ว่าเจองูเห่าด้านในรถเก๋ง อาสากู้ภัยก็เลยได้เปิดประตูข้างหลังทางซ้ายพบเบาะคาร์ซีทวางอยู่เบาะหลัง เจองูเห่าขนาดใหญ่เลื้อยอยู่ใต้เบาะข้างหลังซ้าย ก่อนที่จะงูจะเลื้อยไปใต้เบาะหน้าด้านซ้าย

![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1164956618857926677/4ea1d0b5996c23e6.jpg?ex=65578e65&is=65451965&hm=16cff762e3b81fb1a9b50ba797ef85773a06aa359c9d7a4d292a4132485946c3&=)

ต่อจากนั้นสมัครใจก็ไปเปิดประตูหน้าทางด้านซ้าย แล้วงูเลื้อยหนีย้อนกลับไปข้างหลังเข้าไปซุกภายในใต้เบาะคาร์ซีท ก็เลยเอาเบาะคาร์ซีทลงมาจากรถยนต์ ก็เลยกระทำการถอดน็อตข้างหลังเบาะคาร์ซีทออกมา แล้วก็ฉีดยากันยุงทำให้งูเลื้อยออกมา สมัครใจช่วยเหลือแล้วก็ประชากรดีที่เป็นทหารเข้าทำจับงู ด้วยการใช้บ่วงคล้องคองู ยาวราวๆ 1 เมตร ก่อนนำไปใส่ลงไปภายในถุงปุ๋ย เพื่อนำไปปลดปล่อยคืนสู่ธรรมชาติ

โดย นายประยงค์ เล่าว่า ตนขับขี่รถมาพร้อมกับนางน้อย อายุ 66 ปี ภรรยา เพื่อไปรับบุตรสาวแล้วก็หลานสาว วัย 3 ขวบ ที่ท่าอากาศยาน โดยติดเบาะคาร์ซีทมาด้วย ภายหลังจากรับบุตรสาวแล้ว ตนได้ขับขี่รถออกมาจากท่าอากาศยาน โดยบุตรสาวนั่งเบาะหน้าอุ้มบุตรสาววัย 3 ขวบ ส่วนนางน้อยนั่งเบาะหลัง

![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1164956618539139142/3ae8b642b3d3b417.jpg?ex=65578e65&is=65451965&hm=60ff47205633c6ff4892e47a20edc31fdd3ec09cb31ad4efdce57b48166188f5&=)

พอเพียงขับมาถึงปั๊มน้ำมันหน้าท่าอากาศยาน นางน้อยพูดว่ามีงูขนาดใหญ่อยู่ในรถยนต์และก็เลื้อยผ่านขา ตนเกรงว่างูจะฉกหลานก็เลยเลี้ยวเข้าไปหยุดในปั๊มน้ำมัน และก็ลงจากรถยนต์ ก่อนที่จะแจ้งสมัครใจช่วยเหลือ ซึ่งตนก็ไม่เคยทราบว่างูเข้ารถยนต์มาได้เช่นไร แต่ว่าคาดการณ์ว่า ตนหยุดรถยนต์แล้วเปิดประตูไว้ งูก็เลยเลื้อยเข้ามาในรถยนต์

“โชคดีที่งูไม่ได้ฉกใคร ซึ่งคนโบราณมั่นใจว่า แม้งูเข้ารถยนต์หรือบ้านจัดว่ามาให้ลาภ ซึ่งตนก็จะนำเลขทะเบียนรถยนต์ไปเสี่ยงดวง ซื้อสลากกินแบ่ง เลข 89-98-10-01 เผื่อจะดวงดีถูกรางวัล" นายประยงค์ กล่าว
