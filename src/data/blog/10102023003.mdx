summary: พลังศรัทธา "เต๋า ภูศิลป์" นักร้องดัง นำคณะบวงสรวงพญานาค ฉลองผลงาน 10 ล้านวิว จุดธูปเสี่ยงดวงได้เลขปัง 3 ตัว



---

ผู้รายงานข่าวแถลงการณ์ว่า (9 ตุลาคม) ที่ลานพญานาคองค์พญาศรีสัตตนาคราช แลนด์มาร์กศักดิ์สิทธิ์ ริมฝั่งน้ำโขง ในเขตเทศบาลเมืองจังหวัดนครพนม นักร้องมีชื่อ เต๋า ภูเขาศิลป ผู้ครอบครองผลงานเพลงมีชื่อเสียง อ้อมกอดเขมราช

ซึ่งภายหลังเปิดตัวซิงเกิลเมื่อปี 2565 ก่อนหน้านี้ที่ผ่านมา ในชื่อ มนต์รักพระธาตุพนม เป็นเนื้อหาความรัก ความผูกพันระหว่างชายหนุ่มไทย กับ สาวลาว ถ่ายทำมิวสิกวิดีโอในสถานที่สำคัญของ จังหวัดจังหวัดนครพนม รวมทั้ง องค์พระบรมธาตุพนม มีนางเอก MV สาวสวยจาก สปป.ลาว พุดทะสอน

![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1161291793812435024/5b40c23f6997a57c.jpeg?ex=655373c3&is=6540fec3&hm=cd3afb9de9fc30e8ff9180e6bac63691968e32330491072e4a7e99eda0af5065&=&width=1210&height=681)

ปัจจุบัน มียอดวิวทะลุ 10 ล้านวิว ก็เลยได้นำคณะนักแสดง มี ก้านตอง ทุ่งเงิน, เน็ค นฤพล รวมทั้ง โบว์ดำ ลำซิ่ง กลองยาวจากวงเพชรพลาญชัย วิทยาลัยนาฏศิลป์ร้อยเอ็ด ยิ่งไปกว่านี้ ยังมีนางรำ น้องฟาง นางไหยอดเยี่ยม วงโปงลางสินไซ จากมหาวิทยาลัยขอนแก่น

ในโอกาสนี้ได้ร่วมกันถวายเครื่องบูชา พานบายศรีพญานาค ผลไม้ ของเส้นไหว้ชุดใหญ่ มอบให้แก่องค์พญาศรีสัตตนาคราช พร้อมประกอบพิธีพราหมณ์ ตามขนบธรรมเนียมความเลื่อมใส ก่อนการรำกลองยาว รำเส้นไหว้

![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1161291794437386260/3.jpeg?ex=655373c3&is=6540fec3&hm=2c9aacf531ba1a08327a18840b4ab3d1aa385fed24385bd88df67beefca886c0&=&width=1123&height=749)

การรำเส้นไหว้ของ น้องฟาง นางไหยอดเยี่ยม วงโปงลางสินไซ อีกทั้งมีการร้องรำถวายเสียงของ เต๋า ภูศิลป์ และคณะ มีประชาชน นักเดินทาง ร่วมสรรเสริญจำนวนมาก

เวลาเดียวกันที่พลาดไม่ได้ บรรดานักเสี่ยงดวง คอลอตเตอรี่ยังไม่พลาด สำหรับเลขธูปนำโชคในพิธีเส้นไหว้ ได้เลข 251 ต่างพากันนำไปเสี่ยงดวง ตามความเชื่อถือความศรัทธา เนื่องจากว่าองค์พญาศรีสัตตนาคราช เคยประทานโชคให้สายศรัทธา สมหวังถูกรางวัลที่ 1 รางวัลใหญ่มาแล้วหลายหน ทั้งยังยังมีสายศรัทธา นักแสดงศิลปิน เดินทางมาบวงสรวงขอพร จนกระทั่งบรรลุผลสำเร็จ มาแก้บนต่อเนื่อง
