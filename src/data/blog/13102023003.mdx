---

บรรดาคอสลากกินแบ่งไม่พลาด แห่มาดูปลาไหลสีทองคำทั้งยังลำตัว มีปากรวมทั้งหางสีแดง ฮือฮา เชื่อเป็นบุตรหลานพญานาคมาให้ลาภ เพราะเหตุว่ามีเลขขึ้นที่ลำตัว

วันที่ 6 ต.ค. 2566 เวลา 17.00 น ที่บ้านเลขที่ 116/1 ม.7 ตำบลชากพง อำเภอแกลง จังหวัดจังหวัดระยอง นางสังวาล ศรีลาเกษ อายุ 50 ปี เล่าว่า เพื่อนบ้านของตนเองได้ไปดักปลารอบๆลำคลองแถวบ้านเมื่อเช้านี้ และก็เที่ยวกลับได้นำปลาไหลตัวนี้มาให้ตน ตนมีความเห็นว่าปลาไหลตัวนี้มีลักษณะลำตัวเป็นสีทองคำ ปากแดง หางแดง ความยามโดยประมาณ 30 ซม. ตนก็เลยเอามาให้ชาวบ้านมอง ทุกคนต่างก็ว่ามันแปลกมากมาย
![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1162382824788205678/3.jpg?ex=65576bdd&is=6544f6dd&hm=84228493fec3dcea5829cc7e0f10fa07e6461b9852d61ac85dd2d60b8eab63f0&=&width=422&height=749)

บางบุคคลก็มองเห็นเป็นตัวเลขขึ้นที่แถวๆหัวของตัวปลา บางบุคคลก็มองเห็นเลขที่ลำตัว ทำประชาชนต่างแห่กันมาดูมีความสนใจปลาไหลตัวนี้ ต่างคนก็ต่างตีเลขไปกันต่างๆนาๆมองเห็น 3 ตัวบ้าง 2 ตัวบ้าง สลับกันดูและตีเลขกันบ้างก็เอามือคลำหัวก็ไม่กัด ปลดปล่อยให้ราษฎรคลำหัวลูกเนื้อลูบตัวกันไป


![enter image description here](https://media.discordapp.net/attachments/1158269424156028998/1162382824322633809/2.jpg?ex=65576bdd&is=6544f6dd&hm=1cc73769fc55eada0b72a26a1bfedfcd44a2ce1df308ebb8e04ecbb6d90b9cfa&=)

ตนนั้นดูแล้วปลาไหลตัวนี้น่ารักน่าเอ็นดู ไม่ว่าจะถูกหวยหรือเปล่าถูกก็จะเลี้ยงเอาไว้อย่างแน่แท้ แต่ ปลาไหลทองคำปากแดงที่พบเจอนั้น ประชาชนต่างมั่นใจว่าเป็นบุตรหลานพญานาคมาให้ลาภ โดยยิ่งไปกว่านั้นหวยรัฐบาลงวดที่จะถึงนี้จะต้องมีเลขที่ปรากฏบนตัวปลาไหลทองคำปากแดงแน่นอน
